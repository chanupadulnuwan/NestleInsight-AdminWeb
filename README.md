# Project Admin Web Dashboard

Admin panel for monitoring operations, approvals, reports, and exports.

## Tech Stack
- React
- Tailwind CSS
- Axios
- Charting library (Recharts / Chart.js)

## Features
- Approval workflows
- Monitoring dashboards
- Export management
- Role-based access
- Forecast Based on POS data

## Setup
npm install
npm run dev
